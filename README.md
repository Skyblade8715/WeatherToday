# WeatherToday

WeatherToday is a simple Android application that allows users to search for cities and view current weather conditions along with an hourly forecast.

## Features

- Search for cities with input validation
- Display matching locations from a remote API
- View detailed weather forecast for a selected city
- Hourly forecast starting from the next upcoming hour
- Temperature color adapts to value (cold, moderate, hot)
- Recent search history persisted locally
